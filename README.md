# Tap-Code-Convertor
Converts Tap-Code To Readable English

Instructions
-----

Run `main.lua` through the command prompt to initiate the program.

**Examples Of Acceptable Input**

1.) `23 24`

2.) `.. .. ... .... ... .... . ....`

3.) `... .. 34 42 ... ... 24 ... ... .. ..`

**Output:**

1.) `Hi`

2.) `Good`

3.) `Morning`
